PROJECT OVERVIEW

Airline Management System is a dedicated and highly configurable system for all airlines, which can be easily accessed by all users. It helps the users to book flights without visiting offline booking counters. This system can be accessible by any user from any location at any time. So in this project we will make such a system in which a passenger would be able to view the availability of flights’ details, as per their requirement as well as they can book the flights online and can also cancel the reservation. The administrator manages the passenger booking system and updates the reservation status.
